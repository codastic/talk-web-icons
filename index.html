First of all

Stop using icon fonts!

Or event worse pixel based icons, you know, retina support is a mess and giving the same icon different colors...

To be more precise

If you are on IE9+ stop using icon fonts!

Why?

Because Chris Coyier says so...

Icon fonts have problems

  • Workflow integration is complex
  • Aliasing and sharpness is different in each browser/system
  • Positioning icon fonts is hard
  • If your font doesn’t load you’ll see ugly characters
  • It is againts HTML semantics and accessibility
If there is time talk more about those points, e.g. with examples.
  • There are tools like iconmoon but they are not really integrated into your grunt/gulp workflow. What if you wanna change individual icons? Its a mess.
